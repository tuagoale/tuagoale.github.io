Built an LLM-driven RPG prototype using GPT-4o and RPG Maker to enable context-aware NPC dialogue generation.

Designed prompt templates and game-state interfaces that allowed NPCs to condition responses on player progress, dialogue history, and task context. The project explored adaptive response generation and interactive narrative control.
